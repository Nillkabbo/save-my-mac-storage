# Contributing

Thanks for contributing to macOS Cleaner!

## Development Setup

1. Create a virtual environment:
   
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

## Code Style

- Keep changes minimal and focused
- Prefer clear, readable code over cleverness
- Avoid adding new dependencies unless required

## Testing

- Run any available checks before submitting
- Add tests for new functionality where possible

## Pull Requests

- Describe the problem and solution
- Include relevant screenshots or logs for UI changes
- Keep PRs small and easy to review

## Reporting Issues

- Provide steps to reproduce
- Include macOS version and Python version
- Attach relevant logs when available
